Алгоритм установки:

Активируйте ваше виртуальное окружение! Это критически важно. Плагины, как и все зависимости проекта, должны быть установлены в изолированное окружение.

# Для Windows
venv\Scripts\activate
# Для macOS/Linux
source venv/bin/activate


Используйте pip install:
Например, давайте установим плагин pytest-cov для измерения покрытия, который мы подробно разберем в следующем уроке.

pip install pytest-cov


"Заморозьте" зависимости: После установки нового плагина обязательно обновите ваш файл requirements.txt, чтобы ваши коллеги и CI-сервер тоже установили этот плагин.

pip freeze > requirements.txt


Теперь в вашем requirements.txt появятся новые строчки, например, pytest-cov==... и его зависимости.
